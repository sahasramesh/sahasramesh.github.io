This is a simple Accordion made only with jQuery<

Version: 0.2


Key Features

  • No further Plugins needed
    just jQuery and you're ready to go
  • Progressive Enhanced
    accordion effect doesn't work with disabled javascript but it will not interrupt your layout or either hide any content.
  • Styleable
    just style this accordion the way you want - all neccesary styles for proper working of the accordion are added through js.
