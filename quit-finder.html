quit finder is a Python Program that quits finder. Check out the code or demo.

Finder is notorious for being the only application that can’t be quit on a Mac just by right-clicking the icon in the dashboard. This program allows you to remove the dot below Finder to achieve a cleaner look for your Desktop.

Upon clicking the ‘do it’ button, the program enters a few commands into the terminal that quit Finder. As you can see, the dot below the Finder icon has disappeared, indicating that the application has been terminated.
